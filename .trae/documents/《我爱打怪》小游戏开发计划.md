# 《我爱打怪》小游戏开发计划（Vite版本）

## 技术栈选择
- **构建工具**：Vite 5.x
- **前端框架**：Vue 3.x + Composition API
- **状态管理**：Pinia
- **CSS框架**：Tailwind CSS
- **本地存储**：localStorage + Pinia持久化
- **开发语言**：JavaScript（ES6+）

```javascript
// stores/gameStore.js
import { defineStore } from 'pinia';
import { ref, computed } from 'vue';

export const useGameStore = defineStore('game', () => {
  // 状态
  const isGameStarted = ref(false);
  const currentMap = ref(null);
  const gamePhase = ref('exploration'); // exploration/battle
  
  // 动作
  function startGame() { /* 开始游戏逻辑 */ }
  function endGame() { /* 结束游戏逻辑 */ }
  function enterBattle(monster) { /* 进入战斗逻辑 */ }
  function exitBattle() { /* 退出战斗逻辑 */ }
  
  return {
    isGameStarted,
    currentMap,
    gamePhase,
    startGame,
    endGame,
    enterBattle,
    exitBattle

  persist: true // 持久化到localStorage
});
```

## 实现特点

1. **现代前端技术栈**：使用Vite + Vue 3 + Pinia，开发效率高，性能优秀
2. **响应式设计**：使用Tailwind CSS实现响应式布局，适配不同设备
3. **状态管理清晰**：Pinia状态管理，状态流向明确，便于调试和扩展
4. **模块化设计**：各模块独立，便于维护和扩展
5. **本地数据持久化**：使用Pinia持久化插件，自动保存游戏进度
6. **预留扩展接口**：支持后续添加新角色、技能、地图等内容

## 扩展接口设计

1. **角色扩展**：通过models/Character.js基类扩展新角色类型
2. **技能扩展**：在constants/skills.js中添加新技能数据
3. **怪物扩展**：在constants/monsters.js中添加新怪物数据
4. **地图扩展**：通过Map类的generateEntities方法扩展地图生成规则
5. **UI扩展**：通过Vue组件化设计，便于添加新的UI组件

## 游戏流程

1. **游戏初始化**：加载或创建新游戏
2. **地图生成**：随机生成包含怪物、宝箱和障碍物的地图
3. **角色移动**：玩家通过方向键或点击移动角色
4. **触发事件**：
   - 遇到怪物：进入战斗界面
   - 遇到宝箱：获得奖励
   - 遇到障碍物：无法通过
5. **战斗系统**：回合制战斗，玩家选择技能攻击
6. **战斗结果**：
   - 胜利：获得经验和奖励
   - 失败：游戏结束或回到初始状态
7. **游戏存档**：自动保存游戏进度

## 开发环境搭建

1. **安装Node.js**：v18+或v20+
2. **初始化项目**：
   ```bash
   npm create vite@latest . -- --template vue
   npm install
   ```
3. **安装依赖**：
   ```bash
   npm install pinia pinia-plugin-persistedstate
   npm install -D tailwindcss postcss autoprefixer
   ```
4. **配置Tailwind CSS**：
   ```bash
   npx tailwindcss init -p
   ```
5. **启动开发服务器**：
   ```bash
   npm run dev
   ```

## 构建生产版本

npm run build
