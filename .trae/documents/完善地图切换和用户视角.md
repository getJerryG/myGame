# 地图资源等级系统实现计划

## 问题分析

当前地图资源系统存在以下问题：

1. 地图资源（怪物、宝箱）没有明确的等级属性
2. 不同等级的资源没有差异化的展示方式
3. 资源分布与玩家等级没有关联
4. 资源等级系统缺失，影响游戏可玩性

## 解决方案

1. **添加资源等级属性**：

   * 为怪物和宝箱添加等级属性

   * 等级与玩家等级或地图区域相关联

   * 不同等级资源具有不同的属性和奖励

2. **差异化展示方式**：

   * 根据资源等级显示不同的图标

   * 使用颜色或边框区分不同等级的资源

   * 添加等级标识，让玩家直观了解资源等级

3. **优化资源生成逻辑**：

   * 根据玩家等级生成适合的资源等级

   * 确保资源等级分布合理，随着玩家探索逐渐提升

   * 不同地图类型有不同的资源等级分布

## 修改方案

* 不同地图类型有不同的等级范围

## 关键代码修改

### 1. 为Monster类添加level属性

```javascript
// 怪物角色类
export class Monster extends Character {
  constructor(monsterId) {
    // ... 现有代码 ...
    
    // 怪物特有属性
    this.expReward = monsterData.exp; // 击败后获得的经验值
    this.goldReward = monsterData.gold; // 击败后获得的金币
    this.difficulty = Math.floor((stats.hp + stats.attack) / 20); // 怪物难度
    this.level = this.difficulty; // 怪物等级，基于难度计算
  }
}
```

### 2. 为宝箱添加等级系统

```javascript
// 生成宝箱
spawnChest(x, y) {
  // ... 现有代码 ...
  
  // 添加宝箱等级
  const level = Math.floor(Math.random() * 3) + 1; // 1-3级宝箱
  
  // 根据等级调整奖励
  gold *= (1 + level * 0.2);
  exp *= (1 + level * 0.3);
  
  const chest = {
    id: chestId,
    type: MAP_CONFIG.CELL_TYPES.CHEST,
    position: { x, y },
    chunkId: this.chunkId,
    opened: false,
    level: level, // 宝箱等级
    rewards: {
      gold: Math.floor(gold),
      exp: Math.floor(exp),
      item: this.generateRandomItem(level) // 根据等级生成物品
    }
  };
  
  // ... 现有代码 ...
}
```
