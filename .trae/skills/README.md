## 技能列表
- **code-review** 代码审核
- **bug-fix** 修复 bug
- **feature-add** 添加新功能
- **refactor** 代码重构
- **frontend-design** 前端设计
- **js-doc-generator** 生成 jsDoc 文档
- **requirement-analysis** 需求分析
- **web-search** 网页搜索

## 技能描述
- **code-review** 代码审核：审核其他开发者的代码，确保代码质量和规范。
- **bug-fix** 修复 bug：定位并修复代码中的错误，确保软件正常运行。
- **feature-add** 添加新功能：根据需求分析，添加新的功能模块到软件中。
- **refactor** 代码重构：优化代码结构，提高代码可读性和可维护性。
- **frontend-design** 前端设计：设计用户界面，确保用户交互友好。
- **js-doc-generator** 生成 jsDoc 文档：根据代码中的注释，自动生成 jsDoc 文档。
- **requirement-analysis** 需求分析：分析用户需求，确保软件功能符合用户期望。
- **web-search** 网页搜索：基于MCP（Microsoft Copilot Platform）的网页搜索功能，能够接收用户的搜索查询，通过可靠的网页搜索API执行搜索操作，并以结构化、易读的格式返回相关搜索结果。
